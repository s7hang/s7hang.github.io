这个项目基于 Academic Pages 模板搭建，使用 Jekyll 生成静态页面，并适配 GitHub Pages 部署。

主要内容包括：

- 中文首页与个人简介
- 项目、论文和简历页面
- 可通过 Markdown 持续维护的内容结构
- 适合后续添加论文、项目、PDF 和外部链接的目录约定

后续可以在 `_portfolio/` 目录中继续添加新的项目 Markdown 文件。
